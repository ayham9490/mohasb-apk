# تصميم تطبيق Mohasb - المحاسب البسيط

## نظرة عامة
تطبيق محاسبة شخصية متقدم يدعم إدارة الحسابات والمعاملات المالية مع مزامنة Google Sheets والنسخ الاحتياطي وتصدير PDF بدعم كامل للغة العربية.

## قائمة الشاشات

| الشاشة | الوصف | الوظيفة الرئيسية |
|--------|-------|------------------|
| **Splash Screen** | شاشة البداية | عرض شعار التطبيق لمدة ثانيتين |
| **PIN Screen** | شاشة إدخال PIN | التحقق من رمز PIN للدخول الآمن |
| **Login Screen** | شاشة تسجيل الدخول | تسجيل الدخول أو الانتقال للتسجيل |
| **Register Screen** | شاشة التسجيل | إنشاء حساب جديد (الاسم والبريد) |
| **Home Screen** | الشاشة الرئيسية | عرض ملخص الحسابات والمعاملات الأخيرة |
| **Accounts Screen** | إدارة الحسابات | عرض قائمة الحسابات وإضافة حسابات جديدة |
| **Add Transaction Screen** | إضافة معاملة | إضافة معاملة مالية جديدة |
| **Statement Screen** | كشف الحساب | عرض تفاصيل المعاملات وتصديرها |
| **Settings Screen** | الإعدادات | إدارة PIN والمزامنة والنسخ الاحتياطي |

## المحتوى الرئيسي والوظائف

### Home Screen (الشاشة الرئيسية)
- **المحتوى**: 
  - ملخص الحسابات بعملات مختلفة
  - إجمالي الأرصدة
  - آخر 5 معاملات
  - أزرار سريعة (إضافة معاملة، إضافة حساب، كشف حساب)
- **الوظائف**: 
  - عرض الملخص
  - الانتقال للشاشات الأخرى
  - تحديث البيانات

### Accounts Screen (إدارة الحسابات)
- **المحتوى**: 
  - قائمة الحسابات (الاسم، النوع، الرصيد)
  - أنواع الحسابات: زبون (Customer)، مندوب (Agent)، شركة (Company)
- **الوظائف**: 
  - عرض الحسابات
  - إضافة حساب جديد
  - حذف حساب
  - تعديل الرصيد

### Add Transaction Screen (إضافة معاملة)
- **المحتوى**: 
  - نموذج إدخال (الحساب، المبلغ، العملة، البيان)
  - العملات المدعومة: SYP, USD, EUR, TRY
- **الوظائف**: 
  - إضافة معاملة جديدة
  - حفظ في التخزين المحلي
  - مزامنة تلقائية

### Statement Screen (كشف الحساب)
- **المحتوى**: 
  - قائمة المعاملات مع التفاصيل
  - خيارات التصفية والبحث
  - خيارات التصدير (PDF)
- **الوظائف**: 
  - عرض المعاملات
  - تصدير إلى PDF
  - مشاركة التقارير

### Settings Screen (الإعدادات)
- **المحتوى**: 
  - إدارة PIN
  - مزامنة Google Sheets
  - النسخ الاحتياطي
  - إعدادات الحساب
- **الوظائف**: 
  - تعديل PIN
  - تسجيل الدخول إلى Google
  - مزامنة يدوية
  - تفعيل المزامنة التلقائية

## تدفقات المستخدم الرئيسية

### تدفق الدخول الآمن
1. المستخدم يفتح التطبيق
2. عرض شاشة البداية (Splash) لمدة ثانيتين
3. الانتقال لشاشة إدخال PIN
4. إدخال PIN (الافتراضي: 0000)
5. الانتقال لشاشة تسجيل الدخول

### تدفق إضافة معاملة
1. المستخدم ينقر على "إضافة معاملة" من القائمة السفلية
2. ملء النموذج (الحساب، المبلغ، العملة، البيان)
3. النقر على "حفظ"
4. حفظ المعاملة في التخزين المحلي
5. إذا كانت المزامنة التلقائية مفعلة، يتم المزامنة مع Google Sheets

### تدفق المزامنة مع Google Sheets
1. المستخدم يذهب للإعدادات
2. ينقر على "تسجيل الدخول إلى Google"
3. يتم إنشاء جدول بيانات "Mohasb Backup"
4. يمكن للمستخدم تفعيل "المزامنة التلقائية"
5. عند كل معاملة جديدة، يتم المزامنة التلقائية

### تدفق تصدير PDF
1. المستخدم يذهب لشاشة كشف الحساب
2. يختار نطاق التاريخ (اختياري)
3. ينقر على "تصدير PDF"
4. يتم إنشاء ملف PDF مع دعم كامل للغة العربية
5. يمكن مشاركة الملف عبر التطبيقات الأخرى

## معايير التصميم
- **الاتجاه**: عمودي (Portrait) فقط
- **الاستخدام بيد واحدة**: نعم
- **معايير iOS**: متوافق مع Apple Human Interface Guidelines
- **الخط**: النصوص العربية تستخدم خط مناسب للعربية
- **الهامش**: 20dp على الأطراف
- **التباعد**: 16dp بين العناصر
- **نوع الأيقونات**: Material Icons
- **الظلال والتأثيرات**: تأثيرات طفيفة وحديثة
- **الحركات والانتقالات**: انتقالات سلسة وسريعة

## معايير تجربة المستخدم
- **الاستجابة**: جميع الأزرار والعناصر التفاعلية يجب أن تستجيب بسرعة
- **التغذية الراجعة**: تأثيرات بصرية وهزية عند التفاعل
- **الوضوح**: جميع النصوص يجب أن تكون واضحة وسهلة القراءة
- **الملاحة**: ملاحة بسيطة وسهلة الفهم
- **الأداء**: تحميل سريع وعدم تجميد التطبيق
